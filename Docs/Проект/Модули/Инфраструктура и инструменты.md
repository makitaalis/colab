# Инфраструктура и инструменты

## Системные компоненты

- Сеть: `NetworkManager`, `nftables`, WireGuard.
- Время: `chrony/timesyncd`.
- Логи: `journald`, ротация.

## Операционные инструменты

- Baseline/inventory: `scripts/opizero_baseline.py`, `scripts/opizero_inventory.py`
- Масштабирование: `scripts/fleet_registry.py`, `scripts/fleet_rollout.sh`, `scripts/fleet_commission.py`
- Серверные операции: backup/restore, hardening, health auto-alert timers
- Server security posture: `scripts/server_security_posture_check.sh`, `scripts/install_server_security_posture_timer.sh`
