# Масштабирование и реестр

## Типовые сбои

- Dry-run падает на duplicate `vehicle_id`/`wg_ip`.
- После ротации ключей Central получает `401 unauthorized`.
- Ложные `FAIL` в параллельных resilience тестах.

## Базовые проверки

```bash
python3 scripts/fleet_registry.py validate
python3 scripts/fleet_scale_dry_run.py --target-systems 200
python3 scripts/fleet_api_keys.py sync-server --server-host 207.180.213.225 --server-user alis
```
